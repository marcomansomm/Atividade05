# Threads 

```
Criar threads para fazer a soma das linhas de uma matriz.
E mostrar o resultado final.
```

## Project Clone

```
$ git clone https://github.com/marcomansomm/Atividade05.git
$ cd Atividade05

```

## Compile

```
$ make
```

## Run

```
$ make run
```

## Clean

```
$ make clean
```
